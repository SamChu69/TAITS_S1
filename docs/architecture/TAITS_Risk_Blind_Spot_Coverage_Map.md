## 1. TAITS 的核心假設（總前提）

> **市場的最大風險，不是預測錯方向，
> 而是「在錯的情境下，還以為自己是對的」。**

因此，TAITS 的所有策略與模組，
本質上都是 **「風險盲點感測器」**。

---

## 2. 市場常見「被割的 8 大情境」

TAITS 明確定義：
**只要你在以下任一情境中「沒有防護」，長期一定會被割。**

| 編號 | 被割情境 | 描述         |
| -- | ---- | ---------- |
| R1 | 假突破  | 技術面看多，實為誘多 |
| R2 | 派發盤  | 高檔震盪實為出貨   |
| R3 | 假跌破  | 被洗後才起漲     |
| R4 | 結算操盤 | 期貨 / 選擇權扭曲 |
| R5 | 消息配合 | 利多當出貨工具    |
| R6 | 錯誤時機 | 標的對、時間錯    |
| R7 | 單一敘事 | 只相信一種理由    |
| R8 | 情緒過熱 | 最後一棒風險     |

## 4. 為什麼「少一塊」就會出事？（關鍵說明）

### ❌ 只有技術策略（沒有行為 / 操盤）

→ 死在 R1 / R2 / R8

### ❌ 只有價值投資（沒有時間 / Regime）

→ 死在 R6 / R4

### ❌ 只有消息面（沒有驗證）

→ 死在 R5 / R8

### ❌ 只有單一策略線

→ 死在 R7（最常見）

👉 **你加的每一塊，都是為了避免其中一種「必死情境」。**

## 5. 多決策線在風險地圖中的角色（定錨）

多決策線（Multi-Lane）存在的目的不是分散報酬，而是：

> **避免「單一認知模型」壟斷決策。**

| Decision Lane    | 覆蓋風險              |
| ---------------- | ----------------- |
| 行為線（Wyckoff）     | R1 / R2 / R3 / R8 |
| 價值線（Fundamental） | R6 / R7           |
| 執行線（Short-Term）  | R6 / R8           |
| Regime / Risk 線  | R1～R8（全域）         |
