所有系統架構、策略設計、回測規模、代理數量，**必須以本文件為最高約束條件**。

## 1. 使用者設備總覽（實機）

### 🖥 筆電型號

* **ASUS TUF FX504GE**

### ⚙️ 處理器（CPU）

* Intel® Core™ i7-8750H
* 6 核心 / 12 執行緒
* Base 2.20 GHz / Turbo up to 4.10 GHz

### 🧠 記憶體（RAM）

* 32 GB DDR4
* 系統可用約 31.9 GB

### 🎮 顯示卡（GPU）

* NVIDIA GTX 1050 Ti（4 GB）
* **定位說明**：

  * 僅用於顯示與輕量計算
  * ❌ 不作為深度學習訓練核心
  * ❌ 不依賴 CUDA / 大模型推論

### 💾 儲存裝置（已修正）

| 類型  | 規格                            |
| --- | ----------------------------- |
| SSD | **250 GB（系統 / 程式 / 專案）**      |
| HDD | **750 GB（歷史資料 / 回測結果 / Log）** |
| 總容量 | 約 1 TB                        |

📌 **儲存策略定位**：

* SSD：

  * Python / Cursor / Git / 即時運行模組
* HDD：

  * 歷史 K 線
  * 回測結果
  * 原始資料快取

## 3. TAITS 在此設備上的正式定位

### ✅ 可以穩定執行

* 多策略並行（10～30 條策略線）
* 日頻 / 60 分 / 15 分 / 5 分級別決策
* Rule-based + Behavior-based 策略
* 威科夫 / 技術面 / 資金流 / 消息面融合
* 多 Agent 解讀 + 權重整合
* 零股交易邏輯
* 期貨 / 選擇權 **僅作 Regime / Filter / Risk**

### ❌ 明確不做（架構禁止）

* Tick 級高頻交易
* 深度學習模型訓練
* 需要 GPU 長時間滿載的推論
* 雲端依賴型即時系統（目前階段）

---

## 4. 架構設計強制約束（TAITS 必須遵守）

1. **所有策略必須能在 CPU-only 模式下完成決策**
2. 單次決策循環不可超過數秒
3. 回測與即時模組必須可分離執行
4. 策略數量擴充 ≠ 計算量線性爆炸
5. HDD 為冷資料，SSD 為熱資料（嚴禁混用）

---

## 5. 未來擴充原則（但非現在）

* ✅ 策略線增加（而非頻率增加）
* ✅ 期貨 / 選擇權影響層擴充
* ⏳ GPU 升級 / 雲端部署 → **需單獨決策文件**
