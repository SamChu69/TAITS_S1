# 📘 TAITS_Daily_and_Weekly_Checklist.md
## TAITS 人用操作檢查清單（v1.0）

---

## 🗓 每日檢查（5 分鐘）

### 1️⃣ 今天允不允許交易？
- [ ] Regime 是否允許？
- [ ] MR 是否 < 2？
- [ ] 是否有重大事件？

👉 任一為否 → **NO_TRADE**

---

### 2️⃣ 結構是否清楚？
- [ ] 有沒有派發 / 偽突破？
- [ ] 中樞是否尚未完成？
- [ ] 結構是否看不懂？

👉 看不懂 → **不碰**

---

### 3️⃣ 我是不是想「找機會」？
- [ ] 是否因為無聊？
- [ ] 是否因為看到別人賺？
- [ ] 是否因為怕錯過？

👉 有任何一項 → **立刻停手**

---

## 🗓 每週檢查（15–30 分鐘）

### 1️⃣ 本週有沒有違規？
- [ ] 有沒有破例？
- [ ] 有沒有想加碼？
- [ ] 有沒有想攤平？

👉 有 → 記錄，不檢討績效

---

### 2️⃣ 結構是否有變化？
- [ ] 中樞是否擴張？
- [ ] 是否出現派發徵象？

👉 有 → 降權或移出觀察池

---

### 3️⃣ 我有沒有活得比較輕鬆？
- [ ] 心理壓力是否下降？
- [ ] 是否比較不想亂做？

👉 有 → **系統在正常運作**

---

## 終極提醒（不可刪）

> **你不是每天都該做決定，  
> 但你每天都該避免犯錯。**
