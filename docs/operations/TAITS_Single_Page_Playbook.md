# 📘 TAITS_Single_Page_Playbook.md
## TAITS 每日單頁實戰決策手冊（Trade / Wait / No-Trade）

---

## 0. 使用說明（只看一次）

- 本頁為 **TAITS 每日決策唯一入口**
- 不做預測
- 不討論細節策略
- 只回答三個結果之一：
  - TRADE
  - WAIT
  - NO_TRADE

> 若本頁判定為 NO_TRADE，  
> 則 **所有策略一律不得啟用**。

---

## 1. Step 1｜市場 Gate 快速檢查（先決）

### 1.1 市場 Regime

請選擇當前最符合的一項：

- ⬜ TREND（趨勢明確）
- ⬜ RANGE（區間震盪）
- ⬜ HIGH_VOL（高波動 / 失序）
- ⬜ EVENT / SETTLEMENT（事件 / 結算影響）

**規則：**
- HIGH_VOL / EVENT → 預設偏向 WAIT / NO_TRADE

---

### 1.2 Manipulation Risk（MR）

請評估主力操盤風險等級：

- ⬜ MR-0（低）
- ⬜ MR-1（可控）
- ⬜ MR-2（高）
- ⬜ MR-3（極高）

**硬規則（不可違反）：**
- MR-3 → **NO_TRADE**
- MR-2 → 禁止追價、禁止加碼、僅低曝險

---

### ✅ Gate 判定結果

- ⬜ Gate PASS → 進入 Step 2
- ⬜ Gate FAIL → **NO_TRADE（結束）**

---

## 2. Step 2｜風險狀態與曝險檢查

### 2.1 個人 / 系統風險狀態

- ⬜ NORMAL
- ⬜ CAUTION（近期回撤）
- ⬜ LOCKDOWN（連續失誤 / 情緒偏誤）

**規則：**
- LOCKDOWN → **NO_TRADE**
- CAUTION → 僅允許低倉位

---

### 2.2 今日最大曝險允許

- 單一標的最大倉位：＿＿＿%
- 今日總曝險上限：＿＿＿%
- 是否允許加碼：⬜ 是 ⬜ 否

---

## 3. Step 3｜證據層摘要（不是訊號）

### 3.1 Tier-1｜結構性證據（最高權限）

請勾選是否存在下列任一項：

- ⬜ 派發 / 出貨跡象
- ⬜ 假突破 / 誘多
- ⬜ 結算 / 制度性扭曲
- ⬜ 主力行為異常

**規則：**
- 任一成立 → **偏向 WAIT / NO_TRADE**

---

### 3.2 Tier-2｜基本面 / 中期背景

- ⬜ 基本面支持
- ⬜ 價值合理
- ⬜ 產業趨勢順風

**用途：**
- 決定「值不值得等」
- 不提供進場時點

---

### 3.3 Tier-3｜戰術與執行證據

- ⬜ 技術型態支持
- ⬜ 成交量確認
- ⬜ 執行條件清楚

**限制：**
- 僅在 Gate PASS + MR ≤ 1 時有效

---

### 3.4 Tier-4｜不確定性警示

- ⬜ 波動擴張
- ⬜ 分布不穩
- ⬜ 預測模型不確定性升高

**規則：**
- 只允許降權 / 降倉 / WAIT
- 禁止視為利多

---

## 4. Step 4｜決策線（Lane）狀態確認

今日允許啟用的 Decision Lane：

- ⬜ Lane A｜市場結構 / 行為
- ⬜ Lane B｜價值 / 基本面
- ⬜ Lane C｜執行 / 短線
- ⬜ 無（全部關閉）

---

## 5. Step 5｜最終決策（只能選一個）

在不違反以上任何規則下，選擇：

- 🟢 **TRADE**
  - Gate PASS
  - MR ≤ 1
  - 無 Tier-1 否決
  - 風險可控
- 🟡 **WAIT**
  - 結構未否決
  - 但條件不成熟 / 不確定性高
- 🔴 **NO_TRADE**
  - 任一 Gate / MR / 風險否決成立

---

## 6. TAITS 每日一句話（定錨）

> **今天最好的交易，  
> 可能是不交易。  
> 能選擇 NO_TRADE，  
> 才代表系統真的在保護你。**

---

## 7. 使用規範（不可省略）

- 每個交易日 **必須完整走一次本頁**
- 不得因「看起來很想做」而跳步
- 若違反本頁決策，視為 **人為風控破壞**
