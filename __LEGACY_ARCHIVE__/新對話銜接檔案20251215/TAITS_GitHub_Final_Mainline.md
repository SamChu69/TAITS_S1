## 2. TAITS 是什麼（給第一次看到 repo 的人）

**TAITS 不是：**

* ❌ 單一策略
* ❌ 技術指標合集
* ❌ 自動下單程式
* ❌ 某家券商 API 專案

**TAITS 是：**

> 一套專為「台股市場」設計的
> **市場觀察 → 狀態判定 → 策略治理 → 風險否決 → 驗證回測**
> 的 **系統母體（System Mother Architecture）**

核心思想只有一句話：

> **「先判斷市場在什麼狀態，再決定哪些策略『可以被使用』。」**

7. **寧可錯過，不可重傷**
