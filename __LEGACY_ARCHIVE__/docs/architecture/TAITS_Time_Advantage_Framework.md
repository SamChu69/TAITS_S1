# 📘 TAITS_Single_Page_Playbook.md

## TAITS 單頁實戰決策總覽表（Single-Page Playbook）

---

## 0. 使用說明（先讀）

> **這不是策略表，而是「決策表」。**
> 不回答「怎麼買賣」，只回答「值不值得承擔風險」。

使用時，只需 **由上往下跑一次**。

---

## 1️⃣ 時間優勢 × 雙軌總覽（一句話版）

| 問題      | TAITS 回答           |
| ------- | ------------------ |
| 快人一步在哪？ | **Explore（研究與理解）** |
| 慢半拍在哪？  | **Exploit（出手與風控）** |
| 什麼時候賺錢？ | **市場同意 + 風險可控**    |

---

## 2️⃣ Step-by-Step 決策流程（唯一流程）

```
Step 1 → Explore：這檔是不是「值得等」？
Step 2 → Candidate：是否仍在觀察有效期？
Step 3 → Regime：現在適不適合交易？
Step 4 → Strategy：哪些策略被允許存在？
Step 5 → Consensus：共識是否足夠強？
Step 6 → Risk：風險是否值得承擔？
Step 7 → Outcome：Trade / Wait / No Trade
```

---

## 3️⃣ Explore Track（機會探索）快速檢查表

| 檢查項目    | 問題          | 結果    |
| ------- | ----------- | ----- |
| 消息 / 事件 | 是否有「結構性意義」？ | 是 / 否 |
| 基本面     | 是否改變中長期預期？  | 是 / 否 |
| 產業 / 主題 | 是否非短期噪音？    | 是 / 否 |
| 研究理由    | 能否一句話說清？    | 是 / 否 |

**→ 全部為「是」才可進 Candidate Universe**

---

## 4️⃣ Candidate Universe（候選池）狀態判斷

| 狀態          | 意義        |
| ----------- | --------- |
| Active      | 持續觀察，等待市場 |
| Expired     | 時間過久，移除   |
| Invalidated | 假設被證偽     |

📌 **Candidate ≠ 可以交易**

---

## 5️⃣ Market Regime（最高優先權）

| Regime  | 系統態度          |
| ------- | ------------- |
| 趨勢盤     | 可尋找機會         |
| 盤整盤     | 僅限特定策略        |
| 高波動     | 降頻或空手         |
| 事件盤     | 等確認           |
| 結算 / 控盤 | 幾乎全面 NO_TRADE |

📌 **Regime 否決 → 直接 NO_TRADE（不用看下面）**

---

## 6️⃣ Regime × 策略群組允許表（核心）

| Regime | 允許策略         | 強制關閉 |
| ------ | ------------ | ---- |
| 趨勢盤    | 趨勢 / 回檔 / 動能 | 逆勢   |
| 盤整盤    | 區間 / 均值回歸    | 追價   |
| 高波動    | 無（觀察）        | 幾乎全部 |
| 事件盤    | 驗證型          | 技術單獨 |
| 結算盤    | 防守           | 全部主動 |

---

## 7️⃣ Evidence Tier 快速判斷

| Tier   | 內容             | 權限   |
| ------ | -------------- | ---- |
| Tier 1 | 期貨 / 結構 / 系統風險 | 一票否決 |
| Tier 2 | 技術 / Alpha 主體  | 共識核心 |
| Tier 3 | 消息 / 情緒        | 僅微調  |

📌 **Tier 3 永遠不能單獨觸發交易**

---

## 8️⃣ Consensus（共識）最低標準

| 檢查        | 條件           |
| --------- | ------------ |
| 方向一致      | 多數 Tier 2 同向 |
| Regime 相容 | 無衝突          |
| 反證檢查      | 無重大反向 Tier 1 |
| 風險比       | 不對稱為正        |

**→ 任一不符 = WAIT 或 NO_TRADE**

---

## 9️⃣ Risk & Capital（零股友善）

| 原則  | 說明        |
| --- | --------- |
| 小部位 | 不因信心過高放大  |
| 降頻  | 寧可少做      |
| 先活  | 回撤優先權高於報酬 |

---

## 🔟 最終決策輸出（只有三種）

| 結果       | 定義        |
| -------- | --------- |
| TRADE    | 條件齊全，風險可控 |
| WAIT     | 對象對、時機未到  |
| NO_TRADE | 正確的拒絕     |

📌 **NO_TRADE = 成功決策之一**

---

## 11️⃣ 「該賺的錢」判斷準則

> **只吃市場已同意、但尚未過熱的那一段。**

* 不搶第一段（消息）
* 不賭最後一段（情緒）
* **專注中段（共識）**

---

## 12️⃣ TAITS 一句話操作憲章（最重要）

> **先研究、再等待；
> 市場同意，才承擔風險；
> 不做錯事，比做對事更重要。**
