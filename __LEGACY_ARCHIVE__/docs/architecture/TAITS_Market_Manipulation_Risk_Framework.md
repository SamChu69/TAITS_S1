# 📘 TAITS_Market_Manipulation_Risk_Framework.md

## 主力操盤風險治理架構（Market Manipulation Risk Framework）

---

## 0. 文件定位（必讀）

本文件定義 **TAITS 如何面對「主力操盤／非對稱資金行為」所帶來的系統性風險**。

📌 本文件 **不預測主力、不指控市場、不新增交易策略**
📌 僅用於 **辨識高風險市場行為，並主動降低錯誤決策機率**
📌 屬於 **Risk / Regime 層，具有最高否決權**

---

## 1. TAITS 對「主力操盤風險」的正式定義

> **主力操盤風險，指市場中存在具規模之非對稱資金行為，
> 使價格短期偏離自然供需，導致一般策略訊號失真。**

關鍵不是「誰在操盤」，而是：

* 價格是否被刻意控制？
* 波動是否被人為塑造？
* 市場是否正在「說謊」？

---

## 2. 為什麼主力操盤必須是「風險層」，而不是策略層？

TAITS 明確拒絕以下錯誤作法：

* ❌ 把操盤行為當成進場訊號
* ❌ 把主力視為一定會贏的對象
* ❌ 試圖「跟主力一起做」

正確做法是：

> **把操盤行為視為「會讓大多數策略失效的市場狀態」**

👉 因此它必須存在於 **Risk / Regime 層**。

## 4. 高度疑似主力操盤的「行為組合特徵」

TAITS 不依賴單一訊號，而是**行為組合**。

---

### 4.1 價格 × 成交量失真型

| 行為現象       | 解讀       |
| ---------- | -------- |
| 價格突破但量能不跟  | 誘多 / 假突破 |
| 高檔震盪放大量    | 派發可能性高   |
| 跌破關鍵位卻快速拉回 | 假跌破 / 測試 |

📌 **價格說一套，量說另一套**

---

### 4.2 波動控制型

| 行為現象     | 解讀      |
| -------- | ------- |
| 長時間極窄幅震盪 | 控盤吸籌    |
| 關鍵價位反覆來回 | 洗盤 / 換手 |
| 特定時段異常波動 | 人為干預    |

---

### 4.3 消息配合型

| 行為現象          | 解讀     |
| ------------- | ------ |
| 利多不漲          | 上方供給沉重 |
| 利空不跌          | 下方承接強  |
| 消息發布與價格異動高度同步 | 故事配合行為 |

---

### 4.4 結算／制度性操盤型

| 行為現象          | 解讀    |
| ------------- | ----- |
| 期貨結算前後價格扭曲    | 結算影響  |
| 選擇權大量 Pinning | 價格鎖定  |
| 指數與個股明顯背離     | 拉指數效應 |

## 5. TAITS 的「操盤風險等級」分類

| 等級   | 狀態    | 系統態度            |
| ---- | ----- | --------------- |
| MR-0 | 無明顯操盤 | 正常運作            |
| MR-1 | 輕度疑似  | 降權、提高門檻         |
| MR-2 | 高度疑似  | 大幅降頻            |
| MR-3 | 明確操盤  | **強制 NO_TRADE** |

📌 **MR-3 具有一票否決權**

---

## 6. 操盤風險對 TAITS 各模組的影響

### 6.1 對策略宇宙（Strategy Universe）

* MR-2 以上：

  * 關閉突破、追價、動能型策略
* MR-3：

  * 關閉所有主動進場策略

---

### 6.2 對威科夫模組（Wyckoff Evidence）

* 派發 / UT / UTAD → 直接提升操盤風險等級
* 吸籌狀態 → 只能 Explore，不可 Exploit

---

### 6.3 對時間優勢（Time Advantage）

* 操盤盤 **不追求快**
* 寧可錯過，也不提早下注

---

## 7. TAITS 的「操盤盤應對鐵則」

在 MR-2 / MR-3 狀態下，TAITS **必須**遵守：

1. 不追突破
2. 不加碼
3. 不猜方向
4. 不試單
5. 不因為「看起來很強」而破例

📌 **這些不是保守，是保命**

---

## 8. 為什麼這套框架能「實際提高賺錢機率」？

因為多數重大虧損，來自：

* 假突破
* 高檔派發
* 結算盤扭曲
* 主力洗盤期過度交易

TAITS 的做法是：

> **把「最容易虧錢的市場狀態」
> 直接標記為「不值得承擔風險」**
