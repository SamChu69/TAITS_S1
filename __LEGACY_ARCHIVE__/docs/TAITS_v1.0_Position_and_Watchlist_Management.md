# 📘 TAITS_v1.0_Position_and_Watchlist_Management.md
## TAITS v1.0 持股與觀察清單管理模板（零股適用）

---

## 0. 本文件用途（先看）

本文件是 TAITS v1.0 階段 **唯一合法的人工作業模板**，用來：

- 管理「觀察中的標的」
- 管理「已持有的部位」
- 防止情緒化操作
- 提供 S5 檢討的唯一資料來源

📌 本文件 **不記錄預測、不記錄目標價、不記錄感覺**。

---

## 1. 觀察清單（Watchlist）

> 原則：  
> **標的進得來觀察池，比進得來交易重要**

### 1.1 觀察標的清單

| 股票代碼 | 標的名稱 | 所屬 Lane | 結構狀態 | 是否中樞未完成 | 行為風險 | 操作狀態 |
|--------|--------|---------|---------|--------------|---------|---------|
|        |        | A / B   | 清楚 / 模糊 | 是 / 否        | 低 / 中 / 高 | 觀察中 / 移出 |

📌 規則：
- 結構「模糊」→ 不得交易
- 行為風險「高」→ 直接移出

---

### 1.2 移出觀察池的合法理由（必選）

- [ ] 派發 / 出貨跡象
- [ ] 偽突破成立
- [ ] 中樞擴張過久
- [ ] 結構看不懂
- [ ] 與 Regime 不相容

📌 **沒有合法理由，不得移出或移入**

---

## 2. 已持有部位管理（Position）

> 原則：  
> **持有不是信仰，而是被系統暫時允許**

### 2.1 持股紀錄表

| 股票代碼 | Lane | 持有原因（結構） | R 值 | 倉位比例 | 停損類型 | 當前狀態 |
|--------|------|----------------|-----|---------|---------|---------|
|        | A/B  | 結構完成 / 回測 | 0.25 / 0.5 | 零股 / 小 | 風險 / 結構 | 正常 / 觀察 / 準備出場 |

📌 不填「看好」、「感覺」、「新聞」。

---

### 2.2 出場觸發檢查（每日）

- [ ] 停損是否仍有效？
- [ ] 結構是否破壞？
- [ ] 是否進入派發？
- [ ] MR 是否升高？

👉 任一為是 → **準備出場 / WAIT**

---

## 3. 禁止行為自我檢查（每日必看）

- [ ] 我今天想加碼嗎？
- [ ] 我想攤平嗎？
- [ ] 我想因為小錢多試一次嗎？
- [ ] 我是不是因為怕錯過？

👉 有任何一項 → **今天 NO_TRADE**

---

## 4. 每週自我審計（S5 專用）

- 本週是否有違反 S3 / S4？
- 是否有「用結果合理化過程」？
- 是否有策略或結構被誤用？

📌 **只記錄，不辯解**
