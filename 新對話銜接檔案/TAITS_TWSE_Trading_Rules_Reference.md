# TAITS 台股交易制度參考（TWSE / TPEX）

本文件作為 TAITS 交易制度細則參考，
所有最終否決權仍屬於《TAITS_Risk_and_Compliance.md》。

## 1. 交易時間
- 集合競價：08:30–08:59
- 連續交易：09:00–13:30
- 13:30 後不得委託

## 2. 漲跌幅限制
- 一般股票：±10%

## 3. Tick Size
依 TWSE 規範自動調整：
0.01 / 0.05 / 0.1 / 0.5 / 1 / 5

## 4. 委託單類型
允許：
- 限價
- ROD / IOC / FOK
- VWAP（依券商支援）

禁止：
- 未經 TWSE 承認之方式

## 5. 零股交易
- 09:00–13:30 逐筆交易
- 必須遵守 Tick Size

## 6. 禁止行為
- 操縱市場
- 串價
- 配對交易
- 內線交易

## 7. 風控與熔斷
- 個股熔斷
- 重大公告
- 異常波動 → 必須可自動停止交易

## 8. 券商 API 規範
- 憑證與身分驗證
- 頻率限制
- 簽章
- Key rotation
- Rate limit
