# TAITS — Taiwan Alpha Intelligence Trading System

**台灣阿爾法智能交易系統（TAITS）｜系統母體（Architecture-Locked）｜Regime-First｜Risk/Governance Veto**

## 1. TAITS 是什麼

TAITS（Taiwan Alpha Intelligence Trading System）是一套專為 **台灣市場（TWSE / TPEX / TAIFEX）** 設計的「系統母體」，用來把：

* **市場觀察（Data & Agents）**
* **市場狀態判定（Regime Engine）**
* **策略全集（Strategy Universe）**
* **策略治理（Governance / Permission Gate）**
* **風險否決（Risk & Compliance）**
* **回測與驗證（Validation Pipeline）**

整合成 **可長期演進、可審計、可回放、可落地** 的架構。

TAITS 的核心不是「單一策略」，而是：

> **先判斷市場在什麼狀態（Regime），再決定哪些策略可以被使用。**
> **風控與治理可否決一切。**

## 3. TAITS 的不可違反原則（Architecture Laws）

1. **策略 ≠ 下單**（策略只負責「可用訊號」，不負責送單）
2. **Agent ≠ 策略**（Agent 解讀、彙總與證據，不直接產生策略宇宙）
3. **AI ≠ 唯一真理**（AI 只能輔助，不可單獨做最終結論）
4. **Regime 高於單一訊號**（市場狀態優先於任何技術訊號）
5. **Risk / Compliance 可否決一切**（生存優先）
6. **資料不足 = 不判斷（R10）**
7. **觀察型產品不下單**（期貨/選擇權/融資融券只用來觀察股市）

## 5. TAITS 策略宇宙（策略不是下單）

TAITS 的策略宇宙是「可治理訊號集合」，目前封板為：

* **334 條策略（策略 ID 級）**
* 覆蓋：

  * 結構（纏論/鮑迪克）
  * 主力動機（威科夫）
  * 節奏（GMMA / CBL）
  * 趨勢/突破/拉回
  * 均值/強弱/題材/資金流
  * 期貨/選擇權/融資融券/事件（觀察層，具否決權）

> TAITS 的策略層必須受：
> **Regime（市場狀態）＋Governance（治理）＋Risk（風控）** 三重約束。

## 6. Regime（市場狀態）是 TAITS 的核心

TAITS 定義 R1–R10（市場狀態），其中：

* **R7（結構轉換/輪動）** 是為了處理你重視的：
  **題材輪動、中小型股爆發、供應鏈擴散**
* **R5/R6/R8/R9** 則是為了防止：
  恐慌、事件衝擊、流動性失真、假突破追價

Regime Engine 的輸出具備：

* `regime_id`
* `confidence（置信度）`
* `evidence_bundle（證據包）`
* `override_applied（覆蓋原因）`

且可以被回放與審計。

## 7. 治理與風控（可否決一切）

TAITS 的決策不是「訊號多就下」，而是：

* **治理層（TAITS_06）**：決定策略「能用到哪一級」
* **風控層（TAITS_07）**：必要時直接否決任何策略
* **Regime（TAITS_11）**：決定「現在該用哪一類策略」

> 你永遠保有最終決策權（包含是否自動化），系統只提供可審計的建議與限制。

## 一、TAITS 的實際定位（再次明確，避免誤用）

TAITS（Taiwan Alpha Intelligence Trading System）目前的實際定位為：

- ✔ 一套 **台灣市場專用（TWSE / TPEx / TAIFEX）**
- ✔ **以股票為核心決策標的**
- ✔ 結合技術面、方法論、題材、跨市場觀察的  
  **「交易決策作業系統（Decision Operating System）」**

TAITS **不是**：
- 自動交易系統
- 單一策略框架
- 指標集合器
- 黑盒 AI

> **TAITS 的責任是「解釋市場」，不是「替人下注」。**

## 二、期貨 / 選擇權 / 融資融券的正確角色（非常重要）

在 TAITS 中，下列商品 **一律屬於 Observe-only（僅觀察）**：

- 期貨（Futures）
- 選擇權（Options）
- 融資融券（Credit / Margin）

### 它們在系統中「可以做什麼」
- 判斷 **市場 Regime（多頭 / 盤整 / 風險期）**
- 提供 **Risk Override（風險覆蓋）**
- 調整股票策略的 **權重與出場優先序**

### 它們「永遠不可以做什麼」
- ❌ 不得下單
- ❌ 不得模擬下單
- ❌ 不得成為獨立交易標的

> **TAITS 不做「期貨策略」，只做「期貨影響股票的觀察策略」。**

## 三、TAITS 採用的核心方法論（非策略）

### 1️⃣ 威科夫操盤法（Wyckoff）

在 TAITS 中，威科夫被工程化為：

- 主力行為判斷
- 市場階段識別（吸籌 / 推升 / 派發 / 下跌）
- 假突破、出貨風險提示

📌 威科夫 **不是買賣策略**，而是：
- 證據來源
- 結構解釋工具
- 風險提示因子

---

### 2️⃣ 鮑迪克纏論（工程化纏論）

TAITS 採用的是 **工程化、可計算的纏論版本**，用途為：

- 判斷結構位置
- 辨識背離真偽
- 檢查多級別一致性
- 提高勝率，而非提高交易次數

📌 鮑迪克纏論在 TAITS 中：
- 用於 **權重調整 / 出場優先**
- 不直接產生下單指令

## 四、TAITS 為什麼不只做權值股？

TAITS 明確假設：

> 市場的輪動順序通常是  
> **權值股 → 主線題材 → 次主線 → 供應鏈 → 中小型股**

因此 TAITS 的 Universe **強制三池制**：

1. **穩定池**：權值股 / 高流動標的  
2. **輪動池**：主線、次主線題材  
3. **爆發池**：中小型股、材料、工具、供應鏈尾端  

📌 爆發池：
- ❌ 不可被直接刪除
- ✔ 只能降權或標註風險
